ASSAAS!

Better known as 'Automatic Style Sheets As A Service' is a tool designed to help developers with the trails of converting SASS & SCSS.

Get Your API Key

Login using Paypal OpenID

Direct Conversion

POST an SCSS file to http://assaas.hazan.me/convert and you will be returned the compiled CSS in the response body.

POST /convert?apikey=<Your API Key> HTTP/1.1
Host: assaas.hazan.me
Content-Type: application/x-www-form-urlencoded

------Boundary
Content-Disposition: form-data; name="file"; filename="file.scss"
Content-Type: text/css
------Boundary

Inline Conversion

Embed your SCSS & SASS files directly into your website without needing to compile them!

<link href="http://assaas.hazan.me/inline?url=<Encoded URL>" rel="stylesheet">
Example 
<link href="http://assaas.hazan.me/inline?url=http%3A%2F%2Fassaas.hazan.me%2Fres%2Fscss%2Fmain.scss" rel="stylesheet">

REST API

The REST API allows you to upload any CSS, SASS or SCSS file, where it will then be converted into the other formats and made available for you to download. These files will be stored for you to retreive or update at a later time

GET /resources?apikey=<Your API Key>
POST /resources?apikey=<Your API Key>
GET /resources/:id?apikey=<Your API Key>
POST /resources/:id?apikey=<Your API Key>
PUT /resources/:id?apikey=<Your API Key>
DELETE /resources/:id?apikey=<Your API Key>
